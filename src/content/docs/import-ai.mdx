ถ้าคุณมีข้อมูลลูกค้า กิจกรรม หรือยอดขายอยู่ใน Excel, Google Sheets หรือโน้ตในมือถือ — ไม่ต้องพิมพ์ใหม่ทีละรายการ ให้ AI แปลงให้ แล้วอัปโหลดเข้า SolidPlan ได้เลย 😊

ทางที่ดีคือ **เราเตรียม prompt ให้แล้ว** — คุณไม่ต้องเขียนเอง แค่ copy-paste

## สิ่งที่เราเตรียมให้แล้ว

SolidPlan มี **AI Prompt สำเพาะ** ที่บอก AI ภายนอก (ChatGPT, Gemini, Claude) ว่าต้องแปลงข้อมูลให้อยู่ใน format ไหน — prompt นี้ระบุครบ:

- โครงสร้างไฟล์ที่ต้องการ (JSON หรือ CSV)
- ชื่อ field ทั้งหมดพร้อมความหมายภาษาไทย
- ค่าที่ใช้ได้ (เช่น `type` ต้องเป็น `lead` หรือ `customer`)
- field ไหนจำเป็น ไหนเป็น optional
- กฎสำคัญ เช่น การใช้ `tempId` เชื่อม entity ใหม่ด้วยกัน

## ขั้นตอน

### 1. เข้าหน้านำเข้า → แท็บ "เตรียมข้อมูล"

<div className="screenshot-placeholder">วางภาพหน้าจอ: แท็บเตรียมข้อมูลพร้อม AI Prompt (import-ai/prepare-tab)</div>

ไปที่เมนู **นำเข้าข้อมูล** (อยู่ในเมนู avatar มุมขวาบน) → เลือกแท็บ **"เตรียมข้อมูล"**

### 2. เลือก format และคัดลอก AI Prompt

<div className="screenshot-placeholder">วางภาพหน้าจอ: ปุ่มเลือก JSON/CSV + ปุ่มคัดลอก AI Prompt (import-ai/copy-prompt)</div>

- เลือก **JSON** (แนะนำ — นำเข้าหลายประเภทพร้อมกัน) หรือ **CSV** (ทีละประเภท)
- กด **คัดลอก AI Prompt** — prompt จะไปอยู่ใน clipboard ทันที

### 3. ให้ AI แปลงข้อมูล

เปิด ChatGPT / Gemini / Claude แล้ว:

1. **วาง prompt** ที่คัดลอกมา
2. **แนบข้อมูลของคุณ** — คัดลอกจาก Excel, Google Sheets หรือพิมพ์ข้อมูลดิบ
3. **ส่ง** — AI จะแปลงให้อยู่ใน format ที่ SolidPlan รองรับ

ตัวอย่างที่อาจพิมพ์เพิ่ม:

> "นี่คือข้อมูลลูกค้าของฉันจาก Excel ช่วยแปลงให้หน่อย"

### 4. อัปโหลดไฟล์ที่ได้

<div className="screenshot-placeholder">วางภาพหน้าจอ: แท็บนำเข้า + drop zone (import-ai/upload)</div>

กลับมา SolidPlan → แท็บ **"นำเข้า"** → อัปโหลดไฟล์ → กด **ตรวจสอบไฟล์** → ถ้าผ่าน กด **เริ่มนำเข้าข้อมูล**

> ไม่ต้องกลัวผิด — ระบบตรวจสอบก่อนนำเข้าเสมอ ถ้าผิดจะแจ้งให้แก้ก่อน

## ตัวอย่าง: จาก Excel สู่ SolidPlan ใน 3 นาที

```
ข้อมูลใน Excel:
| ชื่อ      | เบอร์โทร    | สนิทแค่ไหน |
|----------|------------|-----------|
| พี่สมชาย   | 081-234    | เพื่อนสนิท |
| คุณวิภา    | 089-567    | รู้จักกัน  |

    ↓ copy prompt + ข้อมูล → ส่งให้ AI

ไฟล์ที่ AI แปลงให้:
{
  "customers": [
    { "tempId": "c1", "callingName": "พี่สมชาย", "phone": "081-234-5678", "relation": "friend" },
    { "tempId": "c2", "callingName": "คุณวิภา", "phone": "089-567-8901", "relation": "know" }
  ]
}

    ↓ upload → SolidPlan

✅ นำเข้าสำเร็จ — ลูกค้า 2 คนเข้าระบบแล้ว
```

## เคล็ดลับเล็ก ๆ 💡

- **ถ้าข้อมูลมีทั้งลูกค้า กิจกรรม และยอดขาย** เลือก JSON — ระบบเชื่อม entity ให้อัตโนมัติผ่าน `tempId`
- **ถ้า AI แปลงผิด** ไม่ต้องกังวล — ระบบตรวจสอบก่อนนำเข้า และจะแจ้ง field ที่ผิดพร้อมรายละเอียด
- **เริ่มจากลูกค้าก่อน** แล้วค่อยนำเข้ากิจกรรมและยอดขาย — เพราะกิจกรรมและยอดขายต้องเชื่อมกับลูกค้า
