ในฐานะตัวแทนประกันชีวิต คุณมีลูกค้าหลายสิบคน แต่ละคนมีนัดหมาย ความต้องการ และรายละเอียดที่ไม่เหมือนกัน SolidPlan ช่วยจัดระเบียบทั้งหมดไว้ในที่เดียว เพื่อให้คุณมีเวลาไปสร้างความสัมพันธ์กับลูกค้าแทนที่จะต้องมานั่งจำว่า "คนนี้เคยคุยเรื่องอะไรไปบ้าง" 😊

## ภาพรวมระบบ

| เมนู | ช่วยอะไรคุณได้ |

| ตั้งค่า | แก้ไขโปรไฟล์ เปลี่ยนรหัสผ่าน ตั้งค่าใบเสร็จ |

## ก่อนเริ่ม

- มีอีเมลที่ใช้สมัคร SolidPlan
- เปิดเบราว์เซอร์ (Chrome, Safari, Firefox, Edge ใช้ได้หมด)
- อุปกรณ์: iPad, Desktop, โทรศัพท์ — ใช้ได้ทุกอย่าง

## ขั้นตอนการเริ่มต้น

### 1. สมัครสมาชิก

- เปิดหน้าเข้าสู่ระบบ → กด **"สมัครสมาชิก"**
- กรอกอีเมลและรหัสผ่าน → กด **"สมัคร"**
- เปิดอีเมลยืนยันตัวตน → กดลิงก์ในอีเมล
- เข้าสู่ระบบได้เลย!

ดูรายละเอียดเพิ่มเติมที่ [เข้าสู่ระบบ](/login/)

### 2. เพิ่มลูกค้าเข้าระบบ

- เริ่มจากเพิ่มรายชื่อผู้มุ่งหวังและลูกค้าที่มีอยู่
- ยิ่งใส่ข้อมูลเยอะ ระบบยิ่งช่วยคุณได้ละเอียด

### 3. บันทึกกิจกรรมทุกครั้งที่ติดต่อ

- โทรหาลูกค้า → บันทึกไว้
- เจอตัว → บันทึกไว้
- ส่งข้อเสนอ → บันทึกไว้
- ทำไว้ทุกครั้ง ระบบจะช่วยจำแทนคุณเอง

### 4. เช็ค Dashboard ทุกเช้า

- ดูว่าวันนี้มีนัดอะไร สัปดาห์นี้ทำกิจกรรมเพียงพอไหม

## เคล็ดลับเล็ก ๆ 💡

ไม่ต้องใส่ข้อมูลลูกค้าให้ครบทุกฟิลด์ตั้งแต่วันแรก — เริ่มจากชื่อและเบอร์โทรก่อน แล้วค่อยเติมรายละเอียดทีหลังได้เรื่อย ๆ

## พร้อมลุยต่อ ✨

เริ่มจาก [เข้าสู่ระบบ](/login/) แล้วมาสร้างรายชื่อลูกค้าแรกของคุณกันเลย
