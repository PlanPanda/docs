เติมเงิน SolidPlan ง่ายแค่สแกน QR Code — ไม่ต้องกรอกข้อมูลบัตร ไม่ต้องโอนผ่านธนาคาร สแกน จ่าย เสร็จ ใช้ได้ทันที 😊

## วิธีเติมเงิน

### 1. เลือกจำนวนเดือน

<Image src={selectMonths} alt="เลือกจำนวนเดือนที่ต้องการเติม" />

- เลือกจำนวนเดือนที่ต้องการเติม
- ระบบจะคำนวณยอดรวมให้ทันที
- เติมหลายเดือน = ส่วนลดมากขึ้น (ตาม Pricing Tier)

### 2. ใช้โปรโมชัน (ถ้ามี)

<Image src={promoCode} alt="ช่องกรอกรหัสโปรโมชัน" />

- กรอก **รหัสโปรโมชัน** ที่ได้รับ
- กด **"ใช้"**
- ยอดจะลดลงตามส่วนลด

### 3. สแกน QR Code ชำระเงิน

<Image src={qrLarge} alt="QR Code ขนาดใหญ่สำหรับสแกนจ่าย" />

- เปิดแอปธนาคารบนมือถือ
- สแกน **QR Code** ที่แสดงบนหน้าจอ
- ตรวจสอบยอด → ยืนยันการชำระเงิน
- เสร็จแล้ว! ยอดจะเข้าระบบอัตโนมัติ

## ชำระผ่าน PromptPay

ระบบรองรับ **PromptPay** — ใช้แอปธนาคารใดก็ได้ที่รองรับ:

- K PLUS (กสิกรไทย)
- SCB EASY (ไทยพาณิชย์)
- Bualuang mBanking (กรุงเทพ)
- Krungthai NEXT (กรุงไทย)
- และธนาคารอื่น ๆ

## ใช้รหัสเชิญเพื่อน (Team Code)

<Image src={teamCode} alt="ช่องกรอก Team Code ตอนชำระเงิน" />

ถ้ามีเพื่อนแนะนำให้มาใช้ SolidPlan:

- กรอก **Team Code** ที่เพื่อนให้มา
- ทั้งคุณและเพื่อนจะได้รับ **โบนัส** เพิ่มเติม

ดูรายละเอียดการชวนเพื่อนที่ [แชร์ให้ทีม](/team/campaign/)

## เคล็ดลับเล็ก ๆ 💡

เติมเงินทีละ 12 เดือน จะคุ้มกว่าเติมทีละเดือน — เพราะมีส่วนลดตามระดับยอดเติม แถมไม่ต้องคอยจำว่าเดือนไหนต้องเติม
