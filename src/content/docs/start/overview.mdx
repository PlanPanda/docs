ในฐานะตัวแทนประกันชีวิต คุณมีลูกค้าหลายสิบคน แต่ละคนมีนัดหมาย ความต้องการ และรายละเอียดที่ไม่เหมือนกัน SolidPlan ช่วยจัดระเบียบทั้งหมดไว้ในที่เดียว เพื่อให้คุณมีเวลาไปสร้างความสัมพันธ์กับลูกค้าแทนที่จะต้องมานั่งจำว่า "คนนี้เคยคุยเรื่องอะไรไปบ้าง" 😊

## ภาพรวมระบบ

<div className="screenshot-placeholder">วางภาพหน้าจอ: ภาพรวมระบบหน้าแรก (getting-started/dashboard-overview)</div>

| เมนู | ช่วยอะไรคุณได้ |
|------|-----------|
| Dashboard | มองเห็นภาพรวมงานทั้งปีในพริบตา รู้เลยว่าสัปดาห์นี้ทำอะไรไปบ้าง |
| ลูกค้า | จัดการรายชื่อ ดูรายละเอียด บันทึกทุกการติดต่อ ไม่พลาดทุกจังหวะขาย |
| ปฏิทิน | วางแผนนัดหมายล่วงหน้า ไม่ลืมนัดสำคัญ |
| แพ็คเกจ | ดูแพ็คเกจ เติมเงิน ตรวจสอบยอดคงเหลือ |
| ทีมงาน | เพิ่มเพื่อน ดูผลงานกันและกัน สร้างแรงบันดาลใจ |
| ยอดขาย | บันทึกยอดขายและติดตามเป้าหมายรายปี |
| นำเข้า | ย้ายข้อมูลเข้าระบบจากไฟล์ JSON/CSV |
| ส่งออก | สำรองข้อมูลหรือส่งต่อไปวิเคราะห์ได้ทันที |
| ตั้งค่า | แก้ไขโปรไฟล์ เปลี่ยนรหัสผ่าน ตั้งค่าใบเสร็จ |

## ก่อนเริ่ม

- มีอีเมลที่ใช้สมัคร SolidPlan
- เปิดเบราว์เซอร์ (Chrome, Safari, Firefox, Edge ใช้ได้หมด)
- อุปกรณ์: iPad, Desktop, โทรศัพท์ — ใช้ได้ทุกอย่าง

## เริ่มต้นอย่างรวดเร็ว

เริ่มใช้งานได้จริงภายใน 10 นาที → [เริ่มใช้จริงใน 10 นาที](/start/quick-start/)

## เส้นทางการใช้งานรายวัน

1. **เปิด Dashboard ทุกเช้า** — ดูว่าวันนี้มีนัดอะไร สัปดาห์นี้ทำกิจกรรมเพียงพอไหม
2. **จัดการลูกค้า** — เพิ่ม ค้นหา บันทึกกิจกรรม
3. **วางแผนในปฏิทิน** — ตั้งนัด ติดตาม ไม่ลืม

## แนวคิด Relationship-first

SolidPlan ออกแบบมาเพื่อช่วยคุณสร้างและรักษาความสัมพันธ์กับลูกค้า:

- **จดทุกครั้ง** — ไม่ต้องจำเอง ระบบช่วยจำ
- **ติดตามต่อเนื่อง** — รู้ว่าคุยเรื่องอะไรไปบ้าง
- **ไม่พลาดนัด** — ปฏิทินช่วยเตือน

## เคล็ดลับเล็ก ๆ 💡

ไม่ต้องใส่ข้อมูลลูกค้าให้ครบทุกฟิลด์ตั้งแต่วันแรก — เริ่มจากชื่อและเบอร์โทรก่อน แล้วค่อยเติมรายละเอียดทีหลังได้เรื่อย ๆ

## พร้อมลุยต่อ ✨

เริ่มจาก [เข้าสู่ระบบ](/start/login/) แล้วมาสร้างรายชื่อลูกค้าแรกของคุณกันเลย
