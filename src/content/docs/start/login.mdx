ข้อมูลลูกค้าทั้งหมดของคุณเป็นส่วนตัว — แต่ละบัญชีแยกอิสระ ไม่มีใครเห็นข้อมูลของคุณได้ เข้าสู่ระบบแล้วเริ่มงานได้เลย ไม่ต้องตั้งค่าอะไรให้ยุ่งยาก 😊

## สมัครสมาชิกใหม่

<Image src={registerScreenshot} alt="หน้าสมัครสมาชิก" />

ใช้เวลาไม่ถึงนาที — สมัครเสร็จก็เริ่มใช้งานได้เลย

1. ที่หน้าเข้าสู่ระบบ → กด **"สมัครสมาชิก"**
2. กรอก **อีเมล** และ **รหัสผ่าน** → กด **"สมัคร"**
3. เปิดอีเมลที่ระบบส่งมา → กดลิงก์ยืนยันตัวตน
4. กลับมาเข้าสู่ระบบด้วยอีเมลและรหัสผ่านที่สมัครไว้ → เรียบร้อย!

> หากไม่เห็นอีเมลยืนยัน ให้ลองดูในโฟลเดอร์ Spam หรือ Junk

## เข้าสู่ระบบ

<Image src={loginScreenshot} alt="หน้าเข้าสู่ระบบ" />

1. กรอก **อีเมล** และ **รหัสผ่าน**
2. กด **"เข้าสู่ระบบ"** → เข้าสู่หน้า Dashboard ได้เลย

## ลืมรหัสผ่าน

{/* TODO: screenshot login/forgot-password.png */}

ไม่ต้องกังวล — ตั้งรหัสผ่านใหม่ได้ใน 3 ขั้นตอน

1. ที่หน้าเข้าสู่ระบบ → กด **"ลืมรหัสผ่าน?"**
2. กรอก **อีเมล** → กด **"ส่งลิงก์รีเซ็ต"**
3. เปิดอีเมล → กดลิงก์ → ตั้งรหัสผ่านใหม่ → เข้าสู่ระบบได้เลย

## เปลี่ยนรหัสผ่าน (หลังเข้าสู่ระบบแล้ว)

เข้าไปที่ **ตั้งค่า → เปลี่ยนรหัสผ่าน** → ดูรายละเอียดที่ [ตั้งค่าโปรไฟล์](/settings/profile/)

## สลับธีม

สลับระหว่างโหมดสว่าง (Light) และโหมดมืด (Dark) ได้ตลอดเวลา — กดไอคอนจันทร์/ดวงอาทิตย์ที่มุมขวาบนของหน้าจอ ใช้แบบไหนสบายตาก็เลือกได้เลย 😊

## พร้อมลุยต่อ ✨

เข้าสู่ระบบเรียบร้อย → ไปดู [Dashboard](/daily/dashboard/) เพื่อเช็คภาพรวมงานของคุณกันเลย
