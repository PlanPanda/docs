SolidPlan ไม่ล็อกข้อมูลคุณไว้ ต้องการสำรอง หรือเอาไปวิเคราะห์ต่อในเครื่องมืออื่น? ส่งออกได้ทั้ง JSON และ CSV ตลอดเวลา 😊

<div className="screenshot-placeholder">วางภาพหน้าจอ: หน้าส่งออกข้อมูล (export/overview)</div>

## ส่งออกได้อะไรบ้าง

| ประเภท | Format | รายละเอียด |
|--------|--------|-----------|
| **ลูกค้า** | JSON / CSV | รายชื่อและข้อมูลลูกค้าทั้งหมด |
| **กิจกรรม** | JSON / CSV | ประวัติการติดต่อทั้งหมด |
| **ยอดขาย** | JSON / CSV | รายการยอดขายและข้อมูลกรมธรรม์ |
| **ทั้งหมด** | JSON | รวมลูกค้า กิจกรรม และยอดขายในไฟล์เดียว |

## ขั้นตอนการส่งออก

### 1. ดูสรุปจำนวนข้อมูล

<div className="screenshot-placeholder">วางภาพหน้าจอ: กล่องสรุปจำนวนข้อมูล (export/count-cards)</div>

หน้า **ส่งออกข้อมูล** จะแสดงจำนวนลูกค้า กิจกรรม และยอดขาย เพื่อให้คุณเช็กข้อมูลก่อนดาวน์โหลด

### 2. เลือกประเภทและรูปแบบไฟล์

<div className="screenshot-placeholder">วางภาพหน้าจอ: การ์ดปุ่มดาวน์โหลด JSON/CSV (export/download-buttons)</div>

- **JSON** เหมาะสำหรับสำรองหรือย้ายระบบ
- **CSV** เหมาะสำหรับเปิดใน Excel/Google Sheets

กดปุ่มที่ต้องการ แล้วไฟล์จะดาวน์โหลดทันที

### 3. ดูคำอธิบายแต่ละ Field

<div className="screenshot-placeholder">วางภาพหน้าจอ: ส่วนคำอธิบาย field (export/field-reference)</div>

กด **คำอธิบายแต่ละ Field** เพื่อดูความหมายของคอลัมน์ ก่อนนำไปวิเคราะห์หรือส่งให้ทีม

## เคล็ดลับเล็ก ๆ 💡

- ส่งออก JSON เป็นสำรองทุกเดือน เพื่อความสบายใจ
- ส่งออก CSV ของยอดขายไปทำ Pivot Table จะเห็น trend ชัดขึ้น
