การเติบโตของทีมขายเริ่มจากการมีเครื่องมือเดียวกัน เมื่อคุณชวนเพื่อนมาใช้ SolidPlan ทั้งคุณและเพื่อนได้ประโยชน์จากระบบแนะนำเพื่อน 😊

<div className="screenshot-placeholder">วางภาพหน้าจอ: หน้าสถิติแชร์ให้ทีม (team/campaign-overview)</div>

## Team Code คืออะไร

`Team Code` คือโค้ดแนะนำเฉพาะของคุณ (เช่น `SP-XXXX-XXXX`) ที่ใช้ตอนชวนเพื่อนสมัคร

- เพื่อนอาจได้รับโบนัสตอนเติมเงินครั้งแรก
- คุณจะได้ point reward เมื่อเพื่อนทำเงื่อนไขสำเร็จ

> Team Code เป็นระบบชวนเพื่อน ไม่ใช่โครงสร้างสำนักงาน/หน่วย

## วิธีชวนเพื่อน

<div className="screenshot-placeholder">วางภาพหน้าจอ: ปุ่มคัดลอกโค้ดและแชร์ (team/share-code)</div>

1. เปิดหน้า **แชร์ให้ทีม**
2. คัดลอก Team Code หรือแชร์ลิงก์เชิญ
3. ส่งให้เพื่อนผ่าน LINE หรือช่องทางที่สะดวก

## ดูสถิติการชวน

<div className="screenshot-placeholder">วางภาพหน้าจอ: การ์ดสถิติการแนะนำเพื่อน (team/referral-stats)</div>

| ตัวเลข | ความหมาย |
|--------|----------|
| **จำนวนคนที่ชวน** | คนที่สมัครผ่านโค้ดของคุณ |
| **Point ที่ได้รับ** | แต้มสะสมจากการแนะนำ |
| **สถานะรางวัล** | ผ่านเงื่อนไขแล้วหรือยัง |

## เคล็ดลับเล็ก ๆ 💡

- แชร์ Team Code พร้อมบอกประโยชน์ที่เพื่อนได้ จะชวนได้ง่ายขึ้น
- บันทึกผลการชวนไว้ทุกสัปดาห์ เพื่อดูว่าช่องทางไหนได้ผลดี
