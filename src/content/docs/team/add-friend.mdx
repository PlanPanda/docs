ยิ่งมีเพื่อนร่วมทางมากเท่าไหร่ คุณยิ่งมีแหล่งพลังงานบวกให้กระตุ้นตัวเองในวันที่เหนื่อย การเพิ่มเพื่อนใน SolidPlan ง่ายนิดเดียว — แค่มี Team Code หรืออีเมลของอีกฝ่าย 😊

{/* TODO: screenshot team/add-friend-dialog.png */}

## ก่อนเริ่ม

- คุณต้องรู้ **Team Code** ของเพื่อน (ดูได้ที่โปรไฟล์เขา) หรือมี **อีเมล** ของเขา
- เพื่อนที่จะเพิ่มต้องมีบัญชี SolidPlan แล้ว (หรือสมัครผ่านลิงก์ invite)

## วิธีเพิ่มเพื่อน

### 1. เปิดหน้าทีมงาน

- กดเมนู **"ทีมงาน"** ที่ sidebar
- กดปุ่ม **"เพิ่มเพื่อน"** ที่มุมขวาบน

### 2. เลือกช่องทาง

{/* TODO: screenshot team/add-friend-tabs.png */}

มี 2 วิธีให้เลือก:

| ช่องทาง | เมื่อไหร่ใช้ | ผลลัพธ์ |
|--------|-----------|---------|
| **Team Code** | รู้ Team Code ของเพื่อน (เช่น ABC123) | ส่งคำขอ → เพื่อนต้องกดยอมรับ |
| **Email** | รู้อีเมลของเพื่อน | ส่ง invite → เพื่อนคลิกลิงก์ → เป็นเพื่อนกันทันที |

### 3. กรอกข้อมูลและส่ง

**แบบ Team Code:**

- พิมพ์ Team Code ของเพื่อนในช่อง
- กด **"ส่งคำขอ"**
- รอเพื่อนกดยอมรับ

**แบบ Email:**

- พิมพ์อีเมลของเพื่อน
- กด **"ส่งคำเชิญ"**
- ระบบส่งอีเมลไปให้เพื่อน พร้อมลิงก์คลิกรับ

## หลังส่งคำขอแล้ว

{/* TODO: screenshot team/pending-tab.png */}

- **Team Code:** คำขอจะอยู่ในแท็บ **"รอการยืนยัน"** จนกว่าเพื่อนจะกด accept
- **Email:** ถ้าเพื่อนยังไม่มีบัญชี เขาจะได้รับอีเมลเชิญ → คลิกลิงก์ → สมัคร → เป็นเพื่อนกันอัตโนมัติ

## รับคำขอเป็นเพื่อน

{/* TODO: screenshot team/friend-request-badge.png */}

เมื่อมีคนส่งคำขอหาคุณ:

1. ที่หน้าทีมงานจะมีข้อความ **"มีคนชวน X คน"** ด้านบน
2. กดข้อความนั้น → ป๊อปอัปแสดงรายชื่อคนที่ส่งคำขอ
3. กด **"ยอมรับ"** หรือ **"ปฏิเสธ"**

ดูรายละเอียดเพิ่มเติมที่ [จัดการเพื่อน](/team/manage-friends/)

## เคล็ดลับเล็ก ๆ 💡

- Team Code ของคุณอยู่ที่โปรไฟล์ — แชร์ให้เพื่อนได้เลย
- ถ้าส่ง invite ผิดอีเมล ไม่ต้องกังวล ลิงก์จะหมดอายุเองในระยะเวลาหนึ่ง
- ยกเลิกคำขอที่ส่งไปแล้วไม่ได้ แต่ถ้าเพื่อนปฏิเสธ คุณสามารถส่งใหม่ได้
