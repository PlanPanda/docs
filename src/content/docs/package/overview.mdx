SolidPlan มีหลายระดับแพ็คเกจให้เลือก ตามจำนวนลูกค้าและฟีเจอร์ที่คุณต้องการ เลือกแพ็คเกจที่พอดีกับงานของคุณ ไม่ต้องจ่ายเกินความจำเป็น 😊

{/* TODO: screenshot package/package-cards.png */}

## ดูแพ็คเกจทั้งหมด

1. กดเมนู **"แพ็คเกจ"** ที่ sidebar
2. จะเห็นแพ็คเกจทั้งหมดแสดงเป็นการ์ด:
   - ราคาต่อเดือน
   - ฟีเจอร์ที่ได้
   - จำนวนลูกค้าที่รองรับ

{/* TODO: screenshot package/card-details.png */}

## แพ็คเกจปัจจุบัน

{/* TODO: screenshot package/current-balance.png */}

ด้านบนจะแสดง **แพ็คเกจที่คุณใช้อยู่** และ **ยอดคงเหลือ** — รู้ได้ทันทีว่าเหลือเท่าไหร่ ต้องเติมเมื่อไหร่

## เปลี่ยนแพ็คเกจ

{/* TODO: screenshot package/change-package.png */}

1. เลือกแพ็คเกจที่ต้องการ
2. กด **"เลือกแพ็คเกจ"**
3. ระบบจะคำนวณยอดที่ต้องชำระ
4. ดำเนินการชำระเงิน

คุณสามารถอัปเกรดหรือดาวน์เกรดได้ตลอดเวลา — ระบบจะคำนวณยอดต่างให้อัตโนมัติ

## ระบบการหักเงิน

SolidPlan ใช้ระบบ **หักเงินรายวัน** — แต่ละวันจะหักยอดเล็กน้อยตามแพ็คเกจที่ใช้:

- ยอดคงเหลือจะลดลงทุกวันอัตโนมัติ
- เมื่อยอดไม่พอ ระบบจะแจ้งเตือนให้เติมเงิน
- ถ้าไม่เติม แพ็คเกจจะถูกดาวน์เกรดอัตโนมัติ

## เคล็ดลับเล็ก ๆ 💡

เติมเงินล่วงหน้าหลายเดือน — ยิ่งเติมเยอะยิ่งประหยัด เพราะมีส่วนลดตามยอดเติม (Pricing Tiers)

## พร้อมลุยต่อ ✨

เลือกแพ็คเกจแล้ว → ไปดูวิธีชำระเงินกันต่อที่ [การชำระเงิน](/package/payment/)
