ข้อมูลส่วนตัวของคุณคือสิ่งที่ลูกค้าเห็น — เช่น ชื่อที่แสดงบนใบเสร็จ หรือเบอร์โทรติดต่อ อัปเดตให้ถูกต้องไว้ ไม่ใช่เรื่องยาก 😊

{/* TODO: screenshot settings/settings-page.png */}

## แก้ไขข้อมูลส่วนตัว

### 1. เปิดหน้าตั้งค่า

กดเมนู **"ตั้งค่า"** ที่ sidebar

### 2. แก้ไขข้อมูล

{/* TODO: screenshot settings/profile-form.png */}

คุณสามารถแก้ไขได้:

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| ชื่อ-สกุล | ชื่อที่แสดงในระบบ |
| เบอร์โทร | เบอร์ติดต่อของคุณ |
| ภาษา | ไทย / อังกฤษ |

### 3. กดบันทึก

ข้อมูลจะอัปเดตทันที

## เปลี่ยนรหัสผ่าน

{/* TODO: settings/change-password.png */}

1. ที่หน้าตั้งค่า → เลื่อนไปส่วน **"เปลี่ยนรหัสผ่าน"**
2. กรอก **รหัสผ่านปัจจุบัน**
3. กรอก **รหัสผ่านใหม่** (2 ครั้ง เพื่อยืนยัน)
4. กด **"เปลี่ยนรหัสผ่าน"**
5. ระบบจะออกจากระบบให้ → เข้าใหม่ด้วยรหัสผ่านใหม่

## เคล็ดลับเล็ก ๆ 💡

เปลี่ยนรหัสผ่านทุก 3-6 เดือน — ใช้รหัสผ่านที่จำง่ายแต่คนอื่นเดายาก เช่น ชื่อสัตว์เลี้ยง + ตัวเลข + อักษรพิเศษ

## พร้อมลุยต่อ ✨

ข้อมูลส่วนตัวเรียบร้อย → ไปตั้งค่าใบเสร็จกันต่อที่ [ตั้งค่าใบเสร็จ](/settings/receipt-settings/)
