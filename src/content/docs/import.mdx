ถ้าคุณมีข้อมูลลูกค้า ประวัติการติดต่อ หรือยอดขายอยู่ใน Excel/Sheet อยู่แล้ว ไม่ต้องพิมพ์ใหม่ทีละรายการ SolidPlan รองรับการนำเข้าจากไฟล์ JSON และ CSV ให้ระบบจัดการให้ทั้งหมด 😊

<div className="screenshot-placeholder">วางภาพหน้าจอ: หน้านำเข้าข้อมูลทั้งหมด (import/overview)</div>

## นำเข้าได้อะไรบ้าง

| ประเภท | รายละเอียด |
|--------|-----------|
| **ลูกค้า** | รายชื่อ ข้อมูลติดต่อ แท็ก ระดับความสนิท |
| **กิจกรรม** | ประวัติการติดต่อ พร้อมขั้นตอนการขาย |
| **ยอดขาย** | บันทึกการขายและกรมธรรม์ |

> ระบบนำเข้าจะสร้างข้อมูลใหม่ หรืออัปเดตข้อมูลเดิมถ้ามี `id` ตรงกัน ควรตรวจสอบไฟล์ก่อนกดนำเข้าเสมอ

## ขั้นตอนการนำเข้า

### 1. เตรียมไฟล์

<div className="screenshot-placeholder">วางภาพหน้าจอ: แท็บเตรียมข้อมูล (import/prepare-data)</div>

- ที่หน้า **นำเข้าข้อมูล** เลือก format: **JSON** (รวมหลายประเภท) หรือ **CSV** (ทีละประเภท)
- กด **ดาวน์โหลดเทมเพลต** เพื่อดูโครงสร้างที่ระบบต้องการ
- หรือกด **คัดลอก AI Prompt** เพื่อช่วยแปลงข้อมูลให้ตรงรูปแบบ

### 2. อัปโหลดไฟล์

<div className="screenshot-placeholder">วางภาพหน้าจอ: กล่องลากไฟล์นำเข้า (import/drop-zone)</div>

- ไปที่แท็บ **นำเข้า**
- ถ้าเป็น CSV ให้เลือกประเภทข้อมูล (ลูกค้า/กิจกรรม/ยอดขาย)
- ลากไฟล์มาวาง หรือคลิกเลือกไฟล์

### 3. ตรวจสอบไฟล์

<div className="screenshot-placeholder">วางภาพหน้าจอ: ผลตรวจสอบไฟล์นำเข้า (import/validate-result)</div>

กด **ตรวจสอบไฟล์** แล้วอ่านผล:

| ผลลัพธ์ | ความหมาย |
|---------|----------|
| **ตรวจสอบผ่าน** | พร้อมนำเข้า แสดงจำนวนที่จะสร้าง/แก้ไข |
| **ข้อควรระวัง** | มีข้อมูลที่ควรเช็กเพิ่ม แต่นำเข้าได้ |
| **พบข้อผิดพลาด** | โครงสร้างหรือข้อมูลไม่ครบ ต้องแก้ก่อน |

### 4. เริ่มนำเข้าข้อมูล

ถ้าตรวจสอบผ่านแล้ว ให้กด **เริ่มนำเข้าข้อมูล** แล้วรอผลสรุป

<div className="screenshot-placeholder">วางภาพหน้าจอ: สรุปผลนำเข้าสำเร็จ (import/success-summary)</div>

## เคล็ดลับเล็ก ๆ 💡

- เริ่มจาก **ลูกค้า** ก่อน แล้วค่อยนำเข้ากิจกรรมและยอดขาย
- ถ้ามี error หลายรายการ ให้แก้ไฟล์ต้นฉบับแล้วอัปโหลดใหม่
- ใช้ AI Prompt ช่วยแปลงจาก Excel จะประหยัดเวลามาก
