## ปฏิทินกิจกรรมรายปี (Yearly Heatmap) 🗓️

{/* TODO: screenshot dashboard/heatmap.png */}

กราฟแบบ GitHub contribution — ช่องสี่เหลี่ยมเล็ก ๆ แต่ละช่องแทนวันหนึ่งวัน:
- **สีเข้ม** = กิจกรรมเยอะ
- **สีอ่อน** = กิจกรรมน้อย
- **ไม่มีสี** = วันว่าง

**ทำไมต้องดู:** มองเห็นทันทีว่าเดือนไหนทำงานหนัก เดือนไหนว่าง — ช่วยให้คุณวางแผนกระจายงานให้สมดุล ไม่ให้มีช่วงที่ห่างหายจากลูกค้านานเกินไป

## Radar Chart — กิจกรรมตามขั้นตอนการขาย 🎯

{/* TODO: screenshot dashboard/radar-chart.png */}

กราฟวงแหวนแสดงสัดส่วนกิจกรรมในแต่ละขั้นตอน:

| ขั้นตอน | ความหมาย |
|---------|----------|
| ติดต่อ | เริ่มต้นทำความรู้จัก |
| สัมพันธ์ | สร้างความสนิทสนม |
| บริการ | ดูแลและให้บริการ |
| เสนอขาย | เสนอแผนประกัน |
| ปิดการขาย | ปิดดีลสำเร็จ |

**ทำไมต้องดู:** ถ้ากิจกรรมส่วนใหญ่อยู่แค่ "ติดต่อ" แต่ "ปิดการขาย" แทบไม่มี — อาจถึงเวลาปรับกลยุทธ์การตามงานแล้ว เพราะเป้าหมายสุดท้ายคือการปิดการขาย ไม่ใช่แค่ทำความรู้จัก
