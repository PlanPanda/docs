หลังจากคุยกับลูกค้าจบ — ไม่ว่าจะโทร นัดเจอ หรือส่งข้อเสนอ — ใช้เวลาแค่ 2 นาทีเพื่อบันทึกข้อมูลสำคัญ แค่นี้ก็ช่วยให้คุณไม่ลืม และติดตามลูกค้าได้ต่อเนื่องโดยไม่ต้องจำเอง 😊

## ทำไมต้องบันทึกทุกครั้ง?

- **ไม่ต้องจำเอง** — ระบบช่วยจำแทนคุณ 3 เดือนผ่านไปคุณจะไม่จำได้ว่า "คนนี้เคยคุยเรื่องอะไร"
- **ติดตามต่อเนื่อง** — รู้ว่าคุยเรื่องอะไรไปบ้าง จะได้ต่อเรื่องเดิมได้
- **ไม่พลาดนัด** — ตั้งนัดครั้งถัดไปได้ทันที
- **Dashboard ชัดเจน** — ตัวเลขสะท้อนงานจริง เพราะคุณบันทึกทุกครั้ง

## ขั้นตอนหลังคุยลูกค้า

### 1. เปิดหน้าลูกค้า (15 วินาที)

- ไปที่ **ลูกค้า** → เลือกชื่อลูกค้า
- หรือค้นหาชื่อลูกค้าจากช่องค้นหา — พิมพ์ชื่อ เบอร์โทร หรือคำในโน๊ตก็เจอ

### 2. บันทึกกิจกรรม (1 นาที)

1. กด **"+"** ที่ส่วนกิจกรรม
2. กรอกชื่อกิจกรรม เช่น "โทรสอบถามความต้องการ"
3. เลือก **ขั้นตอนการขาย** ที่ตรงกับสิ่งที่ทำ (ดูรายละเอียดที่ [กิจกรรม 5 ขั้นตอน](/help/5-steps/))
4. เลือกวันที่และเวลา

### 3. จดโน้ตสั้น ๆ (30 วินาที)

เขียนสิ่งที่สำคัญ — ไม่ต้องยาว แค่ keyword พอ:

- สิ่งที่ลูกค้าสนใจ
- ข้อกังวลหรือคำถาม
- ข้อมูลส่วนตัวที่ควรจำ (วันเกิดลูก, งานอดิเรก)
- สิ่งที่ต้องส่งต่อ (เอกสาร, ข้อมูล)

### 4. ตั้งนัดครั้งถัดไป (15 วินาที)

- เลือก **วันที่นัดหมายถัดไป**
- จดโน้ตว่าจะทำอะไร (เช่น "ส่งเอกสาร", "โทรติดตาม")
- แม้ยังไม่แน่ใจวันที่ ใส่ไว้ก่อนได้ — แก้ทีหลังง่ายกว่าลืมไปเลย

## ตัวอย่างโน้ตที่ควรจด

### ตัวอย่างที่ 1: โทรหาผู้มุ่งหวัง

```
ขั้นตอน: สัมพันธ์
โน้ต: สนใจประกันสุขภาพ มีครอบครัว 3 คน ภรรยาท้อง 4 เดือน
       ต้องการเปรียบเทียบแผนก่อนตัดสินใจ
นัดถัดไป: 15 ม.ค. — ส่งเอกสารเปรียบเทียบ 3 แผน
```

### ตัวอย่างที่ 2: เจอตัวลูกค้า

```
ขั้นตอน: เสนอขาย
โน้ต: ชอบแผน A แต่กังวลเรื่องเบี้ยประกัน
       ขอคุยกับภรรยาก่อน
นัดถัดไป: 20 ม.ค. — โทรติดตามผล
```

## หลังบันทึกแล้ว

- กิจกรรมจะปรากฏใน [รายละเอียดลูกค้า](/customers/detail/) ทันที
- นัดจะปรากฏใน [ปฏิทิน](/daily/calendar/) ไม่ลืมแน่นอน
- Dashboard จะอัปเดตตัวเลขกิจกรรม เห็นภาพรวมชัดเจน

## เคล็ดลับเล็ก ๆ 💡

- **จดทันที** — ยิ่งนานยิ่งลืม ถ้าขี้เกียจพิมพ์ แค่ keyword สำคัญก็พอ
- **ตั้งนัดทุกครั้ง** — แม้จะยังไม่แน่ใจวันที่ ก็ใส่ไว้ก่อน แก้ทีหลังได้ง่ายกว่าเริ่มต้นใหม่
- **ใช้โน้ตจดรายละเอียดส่วนตัว** — เช่น "ลูกสาวเพิ่งเข้ามหาลัย" พอคราวหน้าไปหา ลูกค้าจะรู้สึกว่าคุณใส่ใจจริง ๆ
