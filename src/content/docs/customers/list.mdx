เมื่อลูกค้ามีหลายสิบคน การหา "คนนั้นคนนี้" ต้องไม่ใช้เวลาเกิน 5 วินาที SolidPlan มีค้นหา กรอง และเรียงลำดับที่ช่วยให้คุณเจอคนที่ต้องการได้ทันที 😊

## ค้นหาลูกค้า

{/* TODO: screenshot customer/search-bar.png */}

- พิมพ์ชื่อ, เบอร์โทร, หรือคำในโน๊ต ที่ช่องค้นหาด้านบน
- ระบบจะแสดงผลลัพธ์ทันทีขณะพิมพ์ — ไม่ต้องกดค้นหา

## กรองรายชื่อ

{/* TODO: screenshot customer/filter-bar.png */}

### กรองตามประเภท

- **ทั้งหมด** — แสดงทุกคน
- **ผู้มุ่งหวัง** — แสดงเฉพาะคนที่ยังไม่ได้ซื้อ
- **ลูกค้า** — แสดงเฉพาะคนที่ซื้อแล้ว

### กรองด้วยตัวเลือกเพิ่มเติม

| ตัวกรอง | ใช้ทำอะไร |
|---------|-----------|
| **แท็ก** | เลือกแท็กที่สนใจ เช่น "VIP", "กอล์ฟ" |
| **ช่วงรายได้** | กรองตามระดับรายได้ |
| **ความสนิท** | สนิทรักเรา / รู้จักพอสมควร / รู้จักแต่ไม่สนิท / ไม่รู้จักเลย |

ใช้ตัวกรองร่วมกันได้ — เช่น กรอง "ผู้มุ่งหวัง" + "รายได้สูง" เพื่อหาคนที่น่าจะเป็นลูกค้ารายใหญ่

## เรียงลำดับ

{/* TODO: screenshot customer/sort-buttons.png */}

กดหัวข้อเรียงลำดับที่ด้านบนรายชื่อ:

| เรียงตาม | เหมาะเมื่อ |
|---------|-----------|
| **ชื่อ** | หาคนตามตัวอักษร |
| **กิจกรรม** | ดูว่าใครมีกิจกรรมเยอะ/น้อย |
| **กิจกรรมล่าสุด** | ดูว่าใครเพิ่งถูกติดต่อ ใครนานไม่ได้ติดต่อ |

กดอีกทีเพื่อสลับเรียงจากน้อยไปมาก / มากไปน้อย

## เลือกลูกค้า

{/* TODO: screenshot customer/customer-card.png */}

- **คลิกที่การ์ด** — เลือกลูกค้าคนเดียว → ดูรายละเอียดด้านขวา
- **ติ๊ก checkbox** — เลือกหลายคนเพื่อดำเนินการพร้อมกัน (เปลี่ยนประเภท, ลบ)

## สิ่งที่แสดงบนการ์ดลูกค้าแต่ละใบ

{/* TODO: screenshot customer/customer-card-annotated.png */}

- รูปโปรไฟล์ (หรืออักษรย่อถ้าไม่มีรูป)
- ชื่อเรียก / ชื่อเต็ม
- เบอร์โทร
- แท็ก (เช่น VIP สีส้ม, แท็กที่กำหนดเอง)
- ไอคอนหัวใจแสดงระดับความสนิท
- สถานะ PDPA
- จุดสีแสดงกิจกรรมล่าสุด

## เคล็ดลับเล็ก ๆ 💡

ลองเรียงตาม "กิจกรรมล่าสุด" จากน้อยไปมาก — จะเห็นทันทีว่าใครที่นานไม่ได้ติดต่อ เหมาะสำหรับวันที่อยากไปหาลูกค้าเก่า ๆ ที่อาจลืมไปแล้ว
