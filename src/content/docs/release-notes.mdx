---

## Release date: 08/June/2026

รอบนี้เราเพิ่มสิ่งที่ช่วยให้คุณทำงานได้ลื่นขึ้น ลดงานจำ และโฟกัสกับลูกค้าได้มากขึ้น ✨

### ไฮไลต์ 🚀

- **นำเข้า / ส่งออกข้อมูล** — ย้ายข้อมูลเข้าออก SolidPlan ได้ทั้ง JSON และ CSV พร้อมใช้ AI ช่วยแปลง format
- **AI Assistant** — ผู้ช่วยส่วนตัวที่คุยกับคุณได้ ช่วยวางแผน สร้างกิจกรรม จัดการลูกค้า
- **หน้าลูกค้าใหม่** — ดูข้อมูลลูกค้าแบบครบจบในที่เดียว เลื่อนดูทุกส่วนได้ง่ายขึ้น

### 📦 นำเข้าและส่งออกข้อมูล

ย้ายข้อมูลเข้าออก SolidPlan ได้สะดวกขึ้น — รองรับทั้ง JSON และ CSV

- **ส่งออกข้อมูล** — ดาวน์โหลดลูกค้า กิจกรรม ยอดขาย เป็น JSON หรือ CSV พร้อมคำอธิบาย field สำหรับ AI
- **นำเข้าข้อมูล** — อัปโหลดไฟล์ JSON หรือ CSV เข้าระบบ มี validation ตรวจสอบก่อนนำเข้า
- **ใช้ AI ช่วย** — ระบบเตรียม AI Prompt และคำอธิบาย field ไว้ให้ copy-paste ไปใช้กับ ChatGPT/Gemini/Claude ได้เลย

→ อ่านวิธีใช้ได้ที่ [ใช้ AI ช่วยนำเข้า](/import-ai/) และ [ใช้ AI ช่วยวิเคราะห์](/export-ai/)

### 🤖 AI Assistant

ผู้ช่วยที่คุยกับคุณได้ — บอกสิ่งที่อยากทำ AI จะจัดการให้

- **พูดคุยได้ตามธรรมชาติ** — พิมพ์บอกสิ่งที่อยากทำ เช่น "เพิ่มนัดคุณสมชายพรุ่งนี้ 10 โมง"
- **จัดการลูกค้า** — ค้นหา เพิ่ม แก้ไขข้อมูลลูกค้าได้จากการพิมพ์
- **วางแผนกิจกรรม** — แนะนำสิ่งที่ควรทำต่อ ช่วยจัดระเบียบ workflow

### 👥 หน้าลูกค้าปรับปรุงใหม่

ดูข้อมูลลูกค้าแบบครบจบ ไม่ต้องกระโดดหน้าไปมา

- **Scroll Spy Navigation** — เลื่อนดูข้อมูลแล้วเมนูด้านข้างจะตามไปอัตโนมัติ
- **สรุปกรมธรรม์** — แท็บใหม่สำหรับดูกรมธรรม์ของลูกค้าในภาพรวม
- **กรองความต้องการประกัน** — เลือกดูลูกค้าตามความต้องการด้านประกัน
- **ข้อมูลส่วนตัวครบขึ้น** — เพิ่มที่อยู่, สถานะ VIP, PDPA consent ในหน้าลูกค้า

### 🤝 ทีมงาน

- **เป้าหมายยอดขายของเพื่อน** — ดู performance และ progress bar ยอดขายของเพื่อนในทีม
- **Monthly Stages Table** — ดูภาพรวมกิจกรรมรายเดือนของตัวแทนในทีม

### 👤 โปรไฟล์

- **รูปโปรไฟล์** — อัปโหลดและลบรูปโปรไฟล์ได้เอง

### 🛠️ ปรับปรุงอื่น ๆ

- เพิ่มปุ่มล้างการค้นหาลูกค้า
- ปรับปรุง layout การค้นหาลูกค้าให้ใช้งานง่ายขึ้น
- ปรับปรุง accessibility ของตัวเลือกลูกค้า

## Release date: 20/May/2026

- **ระบบลูกค้า:** เพิ่มระบบจัดการลูกค้าแบบครบวงจร — รายชื่อ, รายละเอียด, กิจกรรม 5 ขั้นตอน, ไฟล์เอกสาร, แท็ก, ความชอบ
- **Dashboard:** เพิ่ม Yearly Heatmap และ Radar Chart แสดงภาพรวมกิจกรรมตลอดทั้งปี
- **ปฏิทิน:** ระบบนัดหมาย 3 มุมมอง (รายสัปดาห์, รายเดือน, รายปี)
- **ทีมงาน:** เพิ่มระบบเพื่อน, โครงสร้างองค์กร, โปรไฟล์ตัวแทน
- **แพ็คเกจ:** ระบบเติมเงิน QR PromptPay, ประวัติการชำระเงิน, ใบเสร็จ
